SAMPLE A

AI 产品评测样稿

ChatGPT、Claude、DeepSeek：谁更适合辅助长文写作？

  1. 01测试任务：同一篇长文主题，要求三个模型分别生成结构和段落。
  2. 02模型输出对比：比较信息完整度、段落推进和观点明确度。
  3. 03语言风格分析：判断谁更适合冷静说明、谁更适合文学表达、谁更容易空泛。
  4. 04观点提炼：看模型能否从材料里抓出可传播的判断句。
  5. 05使用场景：给出适合长文、短图文、标题优化和复盘的使用建议。
  6. 06最终结论：AI 不替代作者，但能帮助作者更快看见结构问题。

趋势分析样稿

从“会聊天”到“会执行”：AI Agent 为什么会成为下一阶段内容工具？

图文结构

  1. 01现象：用户不再只需要回答，而是希望 AI 帮忙完成任务。
  2. 02趋势：从对话式模型转向可调用工具、可执行流程的 Agent。
  3. 03影响：内容创作者可以把选题、资料、草稿和排期拆成可协作任务。

SAMPLE C

AI 辅助内容生产流程

如何用 AI 把一个模糊想法变成一篇可发布图文？

  1. 01原始想法：先写下不完整但真实的问题。
  2. 02资料收集：整理案例、术语、平台语境和反例。
  3. 03AI 辅助提问：让 AI 帮忙补充角度，而不是直接写成稿。
  4. 04观点提炼：人工筛选最值得表达的判断句。
  5. 05标题优化：测试情绪强度、信息密度和平台点击意图。
  6. 06图文结构：拆成封面、标题页、信息页、观点页、结论页。
  7. 07人工复核：确认事实、语气、立场和传播风险。

FULL SAMPLE / 01

ChatGPT、Claude、DeepSeek：谁更适合辅助长文写作？

测试任务

给三个模型同一个长文主题，要求它们分别生成文章结构、开头段落和标题备选，再比较它们的结构感、语言风格和人工修改成本。

我的判断

我不把模型输出当成最终稿，而是把它们当成不同的写作助手。真正重要的是看它们暴露了哪些结构问题，以及哪些判断仍然必须由人完成。

  1. 01ChatGPT：适合快速搭结构，逻辑推进稳定，但容易显得标准化。
  2. 02Claude：适合长文本语气和段落润色，但需要人工压缩重点。
  3. 03DeepSeek：中文框架生成效率高，适合做初稿和信息压缩。
  4. 04人工复核：最终要重新判断标题、观点密度、事实准确性和平台语气。

FULL SAMPLE / 02

从“会聊天”到“会执行”：AI Agent 为什么会成为下一阶段内容工具？

选题来源

很多人使用 AI 时，已经不满足于“给我一个回答”，而是希望它能帮忙完成一组任务。这个变化让 Agent 成为很适合持续观察的内容方向。

图文方向

这篇更适合做成解释型图文：先解释 Agent 和普通聊天机器人的区别，再给出内容创作者可以使用的场景。

  1. 01第一页：为什么大家开始不满足于聊天式 AI？
  2. 02第二页：Agent 的核心变化是“能执行任务”。
  3. 03第三页：它对内容创作者意味着什么。
  4. 04第四页：哪些任务可以交给 Agent，哪些判断不能。

FULL SAMPLE / 03

如何用 AI 把一个模糊想法变成一篇可发布图文？

流程定位

这不是让 AI 直接写文章，而是把 AI 放进内容生产流程中：它负责帮我扩展问题、整理角度和发现遗漏，人负责判断方向。

适合平台

适合小红书、公众号或作品集中的方法论说明，重点是展示“从想法到发布”的过程。

  1. 01先写下原始问题，不要求完整。
  2. 02让 AI 帮忙追问：这个问题还能从哪些角度看？
  3. 03人工筛选最值得表达的角度。
  4. 04拆成标题、封面、正文结构和结尾判断。
  5. 05发布前做事实、语气、风险和平台适配复核。
