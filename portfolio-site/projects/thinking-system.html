PROJECT 01 / WRITING SYSTEM

思感集｜长期心理观察、内容判断与原创叙事来源

《思感集》是我长期记录心理感受、关系经验、社会观察、身体反应和内容灵感的思考系统。它不是一个孤立的写作项目，而是整个作品集的底层材料库：为 AI 行业选题提供观点来源，为《坠落静默》提供心理叙事基础，也为 IMXS、被单人和展览项目提供情绪与议题判断。

项目类型
长期写作 / 选题系统 / 心理观察 / 内容复盘
对应能力
社会议题观察、观点提炼、长文写作、公开发布、数据判断
使用工具
公众号后台 / 笔记系统 / AI 辅助整理
最终产出
文章、选题库、心理议题库、公众号数据、内容复盘
我的职责
观察记录、主题提炼、写作、发布、复盘、项目转译
阅读建议
先看系统功能，再看选题样本与数据复盘

INPUT SOURCE

选题从持续输入里长出来

心理 / 艺术治疗学习

戏剧治疗、绘画治疗、音乐治疗、阿德勒、荣格、存在主义，为内容提供情绪、自我观察和关系结构维度。

AI 工具与内容观察

观察 AI 工具如何改变写作、创作、信息整理和个体表达，并转化为可讨论的图文选题。

艺术 / 展览 / 文化现场

把展览、城市经验和视觉材料整理为内容角度，而不是停留在“看过”的记录。

日常 / 朋友圈 / 城市经验

雪夜、路灯、廉价食物、散步和朋友圈碎片，为文章提供具体情境和可感知的细节。

SYSTEM VALUE

它不是随笔，而是内容系统

《思感集》的核心价值不在于展示“我写过很多文字”，而在于证明我长期保持观察、表达、复盘和自我修正。它将个人感受转化为可被讨论的公共议题，将心理学习转化为叙事判断，也将日常经验转化为后续项目的选题来源。

NARRATIVE SOURCE

它如何进入《坠落静默》？

《坠落静默》的故事核心并不是由 AI 单独生成，而是建立在长期心理感受和自我观察之上。《思感集》中反复出现的“压抑”“边界”“被看见”“精神崩塌”“自我接纳”“爱与控制”“人与人之间的真实连接”等主题，构成了《坠落静默》的叙事基础。AI 在其中承担扩写、润色和视觉延展工作，最终的主题判断和情绪边界由我完成。

  1. 01长期心理感受沉淀：压抑、逃避、崩塌、自我接纳。
  2. 02关系经验转译：爱与控制、边界感、被看见与真实连接。
  3. 03叙事判断形成：把情绪结构转化为“坠落”和“回到现实”的故事线。
  4. 04AI 辅助展开：扩写段落、整理场景、生成镜头与提示词方向。

QUOTE ARCHIVE

从文字中沉淀出的内容判断

长期表达

“记录的意义远远大于记录的事件本身。”

说明我重视持续记录，而不是只在项目开始后临时找材料。

观点提炼

“文字真是一件伟大的发明。一个善于表达交流的人永远不可能变得偏执。”

说明表达不是装饰，而是理解自己和理解世界的方法。

内容共鸣

“我想做的是给人指引航向的守灯人，而不是告诉别人停在原地是因为你的指南针坏了。”

说明我的内容倾向不是说教，而是陪伴、提醒和照亮。

心理议题判断

“悲伤也好，愤怒也好，焦虑也好，都是我。”

说明我会把情绪当成可以理解的材料，而不是需要被快速消灭的问题。
